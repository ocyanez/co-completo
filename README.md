#  Co-Completo

Sitio web que simula un negocio de venta de completos, desarrollado como proyecto académico.  
La aplicación permite visualizar productos, gestionar un carrito de compras y acceder a información de contacto del negocio.

##  Descripción del proyecto

**Co-Completo** es una página web orientada a la simulación de un comercio gastronómico, enfocada en ofrecer una experiencia simple e intuitiva para el usuario.  
El sitio incluye un catálogo de productos con imágenes, un carrito de compras funcional y secciones informativas como contacto y presentación del negocio.

El objetivo del proyecto fue aplicar conceptos de desarrollo web, organización de contenido y experiencia de usuario.

##  Funcionalidades principales

- Visualización de productos con imágenes y precios  
- Carrito de compras interactivo  
- Agregado y eliminación de productos del carrito  
- Sección de contacto  
- Diseño orientado a una experiencia de usuario clara y sencilla
